# Description
This repository contains a Power BI dashboard created to analyze customer visits, attendance, sales, inventory, and refund data.  
The dashboard is built using Power BI Desktop and visualizes multiple datasets on an overview page.
# Dashboard Features
- Bar chart showing number of visits by each customer.
- Attendance chart displaying Present and Absent counts across multiple days.
- City-based sales slicer (Islamabad, Karachi, Lahore).
- Inventory table listing Product IDs and Product Count.
- Gauge chart highlighting the top-selling product (Bread).
- Donut chart showing total sales by branch.
- Bar chart displaying refund amounts by returned items.
# Data Tables Used
- Attendance  
- Customers  
- Employee  
- Inventory  
- Refund_return  
- Sales

## How to Open
1. Download the `.pbix` file from this repository.
2. Open it in Power BI Desktop (latest version).
3. Navigate through the Overview page and use slicers to filter data.
